# NN-DL — лабораторные по нейросетям и глубокому обучению

Набор Jupyter-ноутбуков по курсу «Нейронные сети и глубокое обучение» (Dank00f edition).  
Здесь живут все эксперименты с нейронами, активациями, градиентами и прочими радостями жизни.
